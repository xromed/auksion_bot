# 📖 ИНСТРУКЦИЯ ПО ЗАПУСКУ БОТА НА macOS

---

## ШАГ 1 — Установи Python

1. Открой **Terminal** (нажми `Cmd + Пробел`, введи `Terminal`, Enter)
2. Введи команду и нажми Enter:
```
python3 --version
```
3. Если видишь `Python 3.x.x` — Python уже есть, переходи к Шагу 2
4. Если пишет "command not found":
   - Зайди на https://www.python.org/downloads/
   - Скачай версию для macOS
   - Запусти установщик `.pkg` и следуй инструкциям

---

## ШАГ 2 — Создай Telegram бота

1. Открой Telegram, найди бота **@BotFather**
2. Напиши ему: `/newbot`
3. Придумай название бота (например: `Мой Аукцион`)
4. Придумай username (например: `moy_auksion_bot`) — должен заканчиваться на `bot`
5. BotFather пришлёт **токен** — длинная строка вида `7123456789:AAF...`
6. Скопируй и сохрани этот токен

### Узнай свой Chat ID:
1. Найди в Telegram бота **@userinfobot**
2. Напиши ему `/start`
3. Он пришлёт твой **ID** (число вида `123456789`)
4. Сохрани это число

## ШАГ 3 — Настрой Google Sheets

### 3.1 Создай проект в Google Cloud:
1. Зайди на https://console.cloud.google.com/
2. Нажми "Select a project" → "NEW PROJECT"
3. Назови проект `auksion-bot` → "CREATE"

### 3.2 Включи нужные API:
1. В меню слева: "APIs & Services" → "Library"
2. Найди **"Google Sheets API"** → нажми "ENABLE"
3. Вернись в Library, найди **"Google Drive API"** → нажми "ENABLE"

### 3.3 Создай ключ доступа:
1. "APIs & Services" → "Credentials"
2. Нажми "CREATE CREDENTIALS" → "Service account"
3. Имя: `auksion-bot` → "CREATE AND CONTINUE" → "DONE"
4. Нажми на созданный аккаунт в списке
5. Перейди на вкладку **"KEYS"**
6. "ADD KEY" → "Create new key" → выбери **"JSON"** → "CREATE"
7. Файл скачается автоматически (вида `auksion-bot-xxxxxxxx.json`)
8. **Переименуй его в `google_creds.json`**

### 3.4 Создай таблицу:
1. Зайди на https://sheets.google.com
2. Создай новую таблицу
3. Назови её точно так: `E-Auksion Лоты`

### 3.5 Дай боту доступ к таблице:
1. Открой файл `google_creds.json` в TextEdit
2. Найди строку `"client_email"` — скопируй адрес (вида `...@...iam.gserviceaccount.com`)
3. Открой таблицу в Google Sheets → нажми **"Поделиться"**
4. Вставь этот email → роль **"Редактор"** → "Отправить"

## ШАГ 4 — Настрой бота

1. Правой кнопкой на файл `bot.py` → "Открыть с помощью" → TextEdit
2. Найди строки в самом начале и замени значения:

```
TELEGRAM_TOKEN   = "ВСТАВЬ_ТОКЕН_БОТА_СЮДА"
TELEGRAM_CHAT_ID = "ВСТАВЬ_СВОЙ_CHAT_ID_СЮДА"
```

Пример как должно выглядеть после замены:
```
TELEGRAM_TOKEN   = "7123456789:AAFxxxxxxxxxxxxxxxxxxxxxxxx"
TELEGRAM_CHAT_ID = "123456789"
```

3. Сохрани файл: `Cmd + S`

---

## ШАГ 5 — Положи файлы в одну папку

Создай папку на рабочем столе `auksion_bot` и положи туда три файла:
- `bot.py`
- `requirements.txt`
- `google_creds.json`

---

## ШАГ 6 — Установи библиотеки

1. Открой Terminal (`Cmd + Пробел` → "Terminal")
2. Перейди в папку с ботом:
```
cd ~/Desktop/auksion_bot
```
3. Установи библиотеки:
```
pip3 install -r requirements.txt
```
Подожди 1-2 минуты пока всё установится.

---

## ШАГ 7 — Запусти бота!

В том же Terminal введи:
```
python3 bot.py
```

Бот запустится, сразу проверит все лоты и пришлёт новые в Telegram.
Потом будет проверять каждый день в **09:00**.

Чтобы остановить бота — нажми `Ctrl + C` в Terminal.

## ⏰ Автозапуск при включении Mac (необязательно)

Чтобы бот запускался сам при каждом включении Mac:

1. В Terminal введи:
```
crontab -e
```
2. Нажми букву `i` на клавиатуре (режим редактирования)
3. Добавь строку:
```
@reboot cd ~/Desktop/auksion_bot && python3 bot.py >> ~/Desktop/auksion_bot/log.txt 2>&1
```
4. Нажми `Esc`, затем напечатай `:wq` и нажми Enter

## ❓ Частые вопросы

**Ошибка "pip3: command not found":**
→ Попробуй: `python3 -m pip install -r requirements.txt`

**Ошибка про Telegram (Unauthorized):**
→ Проверь токен в `bot.py` — скопируй заново от @BotFather

**Ошибка про Google Sheets (SpreadsheetNotFound):**
→ Проверь что название таблицы точно `E-Auksion Лоты`
→ Проверь что поделился таблицей с email из `google_creds.json`
